# Tool Usage Protocol

Before submitting any code answer, you MUST verify your solution using the `code_runner` tool:

1. Call `code_runner` with your implementation as the `code` argument.
2. Check the output — if any test fails, revise and call `code_runner` again.
3. Only submit your final answer after `code_runner` confirms all tests pass.

If `code_runner` is unavailable, state "TOOL UNAVAILABLE" before your answer.
